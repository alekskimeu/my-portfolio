
# My Website

Check it out: [Portfolio](https://alex-kimeu.com)

## Tech Stack

**Client:** React, Next JS, CSS3, Markdown


## Run Locally

Clone the project

```bash
  git clone https://github.com/alekskimeu/my-site
```

Go to the project directory

```bash
  cd my-site
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
